Civil War Photo Sleuth Dashboard

Civil War Photo Sleuth is a platform that leverages leverages the complementary strengths of human vision and facial recognition algorithms to identify unknown soldiers in portraits from the American Civil War Era. Since its release, over 33,000 photos have been identified, with over 18,000 users contributing to the project. To increase user engagement, the team behind Civil War Photo Sleuth wanted to redesign the dashboard in a way that best highlights recent contributions and encourages users to continue contributing to the project.

Background

My Role

I designed the dashboard using Figma and created a high-fidelity prototype using HTML, CSS, and Javascript with guidance from my lab mentors. I then conducted user testing and iterated on the design based on feedback from the lab and some beta testers. Finally, I developed the dashboard using Django before further analyzing results from log data.

Motivation

Increase the visibility of recent user contributions, making it easier and more rewarding for users to contribute to the project.

Problem

Identification efforts are difficult to track in the current interface, with many users only contributing to photos they personally upload.

Solution

Create a dashboard that highlights recent contributions and encourages users to contribute to the identification of other users' photos.

Process Overview

Design

I analyzed the current dashboard and identifying areas for improvement. I then created a low-fidelity prototype using Figma and iterated on the design based on feedback from my lab mentors.

I conducted user testing with the Civil War Photo Sleuth beta testers and lab members isomg a high-fidelity prototype built using HTML, CSS, and Javascript. I further iterated on the design based on this feedback.

Development

I developed the dashboard using Django and deployed it to a staging server. I then conducted user testing with the Civil War Photo Sleuth beta testers and updated the application accordingly over time.

Evaluation

As part of development, I implemented functions to log user interactions with the dashboard such as time spent on the dashboard, amount of scrolling on the dashboard, and number navigation links clicked from the dashboard.

I analyzed the collected log data to determine the effectiveness of the dashboard in increasing user engagement.

Tool Kit

Figma, HTML, CSS, Javascript, Django, Python, PostgreSQL
